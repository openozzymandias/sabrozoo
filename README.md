# SabroZoo – Landing page for pet snacks

Esta es una landing page estática lista para ser desplegada en Vercel.

## Estructura
- `index.html` – página principal.
- `style.css` – estilos responsive y minimalistas.
- `README.md` – este archivo.

## Próximos pasos
1. **Revisar y personalizar**
   - Cambiar el `action` del formulario por tu endpoint de captura de leads (Formspree, ConvertKit, etc.).
   - Sustituir `your-form-id` por el ID real.
   - Ajustar colores, logotipo o imágenes según tu branding.
2. **Conectar con GitHub** – ya está configurado para hacer push.
3. **Desplegar en Vercel** – conecta el repo a Vercel y se publicará automáticamente.
